# CSAO Recommendation System

- Built a Contextual Smart Add-On (CSAO) recommendation system.
- Goal is to predict whether a user will accept a recommended add-on item.
- Created a synthetic dataset with user, cart, and contextual features.
- Performed feature engineering and data preprocessing.
- Trained a Random Forest model.
- Used GridSearchCV for hyperparameter tuning.
- Evaluated the model using Accuracy and ROC-AUC.
- Achieved an AUC score of around 0.75.

Tools Used:
- Python
- Pandas & NumPy
- Scikit-learn
- Matplotlib & Seaborn
